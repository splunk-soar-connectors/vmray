## Port Information

The app uses HTTP/HTTPS protocol for communicating with the VMRay Server. Below are the default
ports used by Splunk SOAR.

| Service Name | Transport Protocol | Port |
|--------------|--------------------|------|
| http         | tcp                | 80   |
| https        | tcp                | 443  |
